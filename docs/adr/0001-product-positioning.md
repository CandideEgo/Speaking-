# ADR-0001: 产品定位 — 视频词汇学习 + 社区 UGC 并重

- **Status**: Accepted — 2026-07-03
- **Supersedes**: 原"AI 口语教练"定位（CLAUDE.md 顶部 Project 段落）

## Context

App 原定位是"AI-powered English speaking practice app"——核心是 AI 发音反馈 + rubric 评分 + SM-2 词汇复习。用户决定**砍掉全部 AI 评分**（见 ADR-0002）。砍掉评分后，app 的核心价值循环必须重新定义，否则产品失去主线，重设计也无方向。

## Decision

核心价值循环重定义为**两条腿并行**：

1. **视频词汇学习**：双语字幕（WhisperX）+ ECDICT 考试词汇标注 + SM-2 间隔复习 + AI 词注释预热。这是学习引擎。
2. **社区 UGC**：创作者中心让用户贡献视频内容，经管理员审核后发布到社区 feed。这是内容引擎。

**口语录音**降为"自我对照"小功能（录一句→回放→跟原声对比），无 AI 评分、无反馈、无留存。SM-2 词汇复习 + 社区保留为核心模块。

## Consequences

- 产品从"AI 口语教练"转向"视频学习社区"。落地页营销文案、首页价值主张需更新。
- 口语练习不再是卖点；录音无反馈闭环，用户自评。
- 重设计重点转向字幕学习 / 词汇 / 浏览 / 创作者 / 社区体验（见 ADR-0005）。
- SM-2 词汇模块 + 社区模块**不动**，是保留的核心。
- `speaking_attempts` 等评分数据冻结（见 ADR-0002、ADR-0003）。
